\# MyStore - Store Management System



\## Overview



MyStore is a Store Management System developed using ASP.NET Core Web API and SQL Server. The project provides a secure and scalable solution for managing customers, products, categories, suppliers, and orders. It follows a layered architecture using Repository Pattern, Service Layer, DTOs, and Entity Framework Core.



\## Features



\* User Registration and Login

\* JWT Authentication and Authorization

\* Customer Management (CRUD Operations)

\* Product Management

\* Category Management

\* Supplier Management

\* Order Management

\* Entity Framework Core Integration

\* SQL Server Database

\* Repository Pattern Implementation

\* DTOs for Data Transfer

\* Swagger API Documentation

\* Clean and Scalable Architecture
